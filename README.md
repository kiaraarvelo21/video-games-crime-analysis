# Video Game Sales and Crime Rates

This project explores whether there is a relationship between video game sales and crime rates across different regions.

## Research Question
Is there a relationship between video game sales and crime rates in North America, Europe, and Japan?

## Data
The analysis combines:

- Regional video game sales data
- Crime rate data across multiple years

The datasets were cleaned and aligned for comparison.

## Methodology
- Data cleaning and preparation
- Regional aggregation of sales data
- Comparative trend analysis
- Exploratory visualization

## Results
The analysis explores whether increases in video game sales correspond with changes in crime rates. The findings highlight the complexity of interpreting correlations in large-scale societal data.

- Pandas
- Matplotlib
- Jupyter Notebook
